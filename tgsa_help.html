Quick Start
Open a FITS image taken using the transmission grating spectrometer (TGS) by selecting File -> Open and choosing the file. Click on the image to center the crosshair on the star in the image. Check that the red selection box in the image is around the diffracted spectrum. Choose Image -> Create Plot to produce a spectrum plot.

Using FITS Images
Use the zoom options to study the image in detail to better position the selection box. Detailed information about the image can be found in the FITS header, which can be revealed by selecting Image -> Show/Hide FITS Header. Compensate for rotation in the spectrum by using the tilt up/tilt down controls. Wavelength accuracy in plots can be improved by comparing position of emission/absorption lines to predicted values and fine-tuning the position of the selection box in the image.

Modifying Plots
Appearance of plots, as well as plotting of spectral lines, can be configured using the options in the Plot menu. Note: modifying the median average width will alter the spectrum data in the plot.

Saving Data
Plots can be saved as PDF files by choosing File -> Save. Raw data from plots can be saved as simple comma-separated-value (CSV) files by selecting File -> Export Data.

Stitching Spectra
In order to compensate for blurring caused by the curved focal plane of the transmission grating, several images can be taken at different focus positions, plotted and combined. Produce spectrum plots of each image in the set and select Plot -> Stitch Spectra... Select the plots from the list on the left and click Add to append them to the list of plots to combine. Plots will be combined in the order in which they appear in the list on the right. You can enter a comma-separated list of wavelengths at which to stich the spectra together, or allow them to be split at even intervals. Press OK to produce a combined plot.

Glossary
CCD
Charged Coupled Device - a 2D array of silicon pixels used to detect incoming photons in order to reproduce a digital image.
FITS image
Flexible Image Transport System - an image format that can contain 2D and 3D data, as well as attached text with information about the image.
transmission grating
a thin plate of glass etched with parallel lines, which diffracts light passing through it into a spectrum
